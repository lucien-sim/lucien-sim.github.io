Predicting EPA inspection outcomes

A machine learning model to predict the probability that an inspection of an air pollution source will reveal a violation of the Clean Air Act. The model is trained on inspection records and other data from the EPA website. Predictions are available through a web application (documentation).

Toy catastrophe model

A toy catastrophe model for flooding in the town of Phoenicia, NY, built from scratch using geographical information and historical stream gauge data. A detailed write-up and demonstration of how the model works is available here. The rest of the code is available on GitHub.

An interactive tool for visualizing CloudSat observations over the Arctic. A limited version of the tool is available through a web application. The application code and instructions for setting the application up on your own machine are available on GitHub.
